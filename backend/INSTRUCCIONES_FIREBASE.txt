📌 PASO 1: Abrir Firebase Console
   └─ URL: https://console.firebase.google.com/
   └─ Selecciona tu proyecto: mision3d-72b4a

📌 PASO 2: Ir a Configuración del Proyecto
   └─ Click en el ⚙️ (engranaje) en la barra lateral
   └─ Selecciona "Configuración del proyecto"

📌 PASO 3: Ir a Service Accounts
   └─ Click en la pestaña "Cuentas de servicio"
   └─ Deberías ver "Firebase Admin SDK"

📌 PASO 4: Generar Nueva Clave
   └─ Asegúrate que esté seleccionado "Node.js"
   └─ Click en "Generar nueva clave privada"
   └─ Confirma en el diálogo que aparece
   └─ Se descargará un archivo JSON automáticamente

📌 PASO 5: Renombrar y Mover el Archivo
   El archivo descargado tendrá un nombre como:
   mision3d-72b4a-firebase-adminsdk-abc12-def456789.json
   
   Acciones:
   ✓ Renómbralo a: firebase-credentials.json
   ✓ Muévelo a: C:\Users\user1\Downloads\mision3d_cart_v2\backend\
   
   Ruta final completa:
   C:\Users\user1\Downloads\mision3d_cart_v2\backend\firebase-credentials.json

📌 PASO 6: Verificar el Contenido
   Abre el archivo y verifica que tenga estos campos:
   ✓ type: "service_account"
   ✓ project_id: "mision3d-72b4a"
   ✓ private_key: "-----BEGIN PRIVATE KEY-----..."
   ✓ client_email: "firebase-adminsdk-...@mision3d-72b4a.iam.gserviceaccount.com"

📌 PASO 7: Iniciar el Servidor
   En la terminal PowerShell:
   
   cd C:\Users\user1\Downloads\mision3d_cart_v2\backend
   npm start
   
   Deberías ver:
   ✅ Credenciales Firebase encontradas
   🔥 Firebase Admin inicializado
   Backend escuchando en puerto 3000
